## Installation
```bash
sudo pip3 install antlr4-python3-runtime
```

## Manual Commands
```bash
python3 run.py gen

python3 run.py test LexerSuite
python3 run.py test ParserSuite
```

## Shortcut Commands
```bash
## For remove all file generated
./clean.sh

## For run test
## Please read and change for your testing purpose
./all.sh
```
